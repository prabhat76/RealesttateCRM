# Propella Real Estate CRM

A modern Angular application replicating the Propella Real Estate CRM design with a clean dashboard interface for managing properties, leads, and deals.

## Features

- **Dashboard Overview**: Statistics cards showing key metrics
- **Property Management**: Recent properties with detailed information
- **Lead Management**: Contact management with status tracking
- **Responsive Design**: Clean, modern interface
- **Navigation**: Sidebar navigation with multiple sections

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open your browser and navigate to `http://localhost:4200`

## Project Structure

- `src/app/components/sidebar/` - Navigation sidebar
- `src/app/components/header/` - Top header with search and user actions
- `src/app/components/dashboard/` - Main dashboard with stats and content
- `src/styles.css` - Global styles and utility classes

## Technologies Used

- Angular 17 (Standalone Components)
- TypeScript
- CSS3 with Flexbox/Grid
- Font Awesome Icons
